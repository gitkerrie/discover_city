动态探索地图 MVP 需求文档

🎯 产品目标

帮助用户以“探索”的方式发现国内冷门/特色旅行目的地，而非传统搜索。

用互动地图+AI推荐卡片增强探索感和趣味性。


👤 目标用户

喜欢旅行、想找新鲜目的地的年轻用户。

已经去过大部分热门城市、想探索冷门地点的资深旅行者。

喜欢视觉化、交互体验的用户。


🔑 核心功能（MVP版本）

1. 首页动态地图

展示中国地图（平面或3D），带缩放和拖拽功能。

地图上的城市/景点点位可点击，采用亮点或波纹效果。

支持简单兴趣标签切换（如美食、文化、自然），动态点亮对应目的地。

2. 灵感卡片弹窗

点击地图上的点位，弹出卡片：

城市/景点名称

一句话推荐理由

代表图片

支持“收藏”按钮，添加到用户的“想去清单”。

3. 兴趣筛选标签

提供3个基础标签：

美食：点亮特色美食城市

文化：点亮文化历史名城

自然：点亮自然风光点

用户点击后地图上动态切换点位显示。

4. 基础收藏功能

用户可收藏喜欢的目的地，收藏信息保存在浏览器本地存储（MVP阶段无需注册登录）。


🖼️ UI/交互简述

地图采用深色背景，点位采用亮色渐变，突出探索感。

标签按钮悬浮在地图顶部，点击时有动画反馈。

点位被点击后，地图可自动聚焦并弹出卡片。

灵感卡片采用卡片翻转或弹出动画。


🏗️ 技术选型（建议）

前端框架：Vue 3 或 React

地图技术：Mapbox GL JS（轻量+好看）

数据存储：JSON文件或简单API（MVP阶段静态数据即可）

AI内容：先用人工撰写的推荐文案，后续接入AI生成。

动画：CSS3 + Mapbox自带动画API。


📦 数据需求（MVP示例）

至少50个城市/景点点位。

每个点位包含：

经纬度

名称

标签类别（美食/文化/自然）

一句话推荐

图片URL


🚀 MVP用户路径

用户进入网站 → 看到可缩放的中国地图 → 默认显示所有点位。

用户切换标签（美食/文化/自然） → 地图动态高亮相应城市。

用户点击点位 → 地图聚焦 → 弹出灵感卡片。

用户点击收藏 → 存入“想去清单”。

用户可关闭卡片继续探索地图。
