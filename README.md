# 动态探索地图

🗺️ 一个帮助用户以"探索"方式发现国内冷门特色旅行目的地的交互式地图应用

## 功能特色

- 🎯 交互式中国地图，支持缩放和拖拽
- 🏷️ 兴趣标签筛选（美食、文化、自然）
- 💫 点击点位弹出灵感卡片
- ❤️ 收藏功能，保存想去清单
- 🎨 精美的动画效果和现代化UI

## 技术栈

- **前端**: 原生HTML + CSS + JavaScript
- **地图**: Mapbox GL JS
- **数据**: 静态JSON文件
- **存储**: 浏览器本地存储

## 快速开始

1. 克隆项目到本地
2. 在项目根目录运行：
   ```bash
   python -m http.server 8000
   ```
3. 打开浏览器访问：http://localhost:8000

## 项目结构

```
Travel/
├── index.html          # 主页面
├── css/
│   └── style.css      # 样式文件
├── js/
│   ├── app.js         # 主应用逻辑
│   └── data.js        # 静态数据
└── assets/            # 图片资源
```

## 使用说明

1. 进入网站查看中国地图
2. 使用顶部标签切换不同兴趣类别
3. 点击地图上的点位查看详情
4. 收藏感兴趣的目的地到想去清单
