# 寻味中国

一张专注中国城市风味的互动地图。首发收录 12 座美食城市，用户可以按城市或招牌美食搜索、查看风味详情、收藏想吃清单，并分享可直接打开城市详情的链接。

## 首发城市

- 经典城市：成都、西安、广州、长沙、武汉、重庆
- 宝藏城市：潮州、柳州、延吉、台州、喀什、扬州

每座城市包含风味简介、3-5 道招牌美食、口味标签和一条觅食提示。

## 本地运行

```bash
npm run dev
```

访问 `http://localhost:8000`。也可以直接运行：

```bash
node scripts/server.js
```

应用使用 Leaflet 和高德地图瓦片，完整显示地图需要网络连接。

## 校验

```bash
npm run check
```

该命令检查 JavaScript 语法，并验证城市数量、字段、坐标、招牌美食和本地图片是否完整。

## 项目结构

```text
.
├── index.html              # 页面结构
├── css/style.css           # 桌面与移动端样式
├── js/app.js               # 地图和界面交互
├── js/data.js              # 12 城美食数据
├── assets/cities/          # 城市主图与来源说明
└── scripts/                # 静态服务器与数据校验
```

收藏数据保存在浏览器 `localStorage` 中，无需注册或登录。图片许可与来源见 `assets/cities/SOURCES.md`。
